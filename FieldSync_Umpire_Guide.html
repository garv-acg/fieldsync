Section 1

How Umpire Scheduling Works

Auto-Schedule algorithm

Click ⚡ Auto-schedule to fill the entire week in one click. The engine selects umpires in this order:

1
Worker is available that day with the umpire role
2
No approved time-off request covers that date
3
Highest seniority (Years Exp) preferred
4
Fewest assignments this week (load balancing)
5
Not already assigned to another game at the same time
Auto-Schedule never overwrites a slot you already filled manually.

Single vs. dual umpire

FieldSync reads the division name to determine how many UMP slots to show on each game card.

DivisionSlots
Majors, AAA, AA, Softball 10U+UMP 1 + UMP 2
Minors, A, Coach Pitch, Tee BallUMP 1 only

A purple "2 umps" badge marks dual-umpire game cards. Both slots must be filled before publishing.

Manual assignment

Use the UMP 1 and UMP 2 dropdowns directly on any game card to assign or override. Set to None to clear. Changes save instantly.

Section 3

Conflicts & Availability

Conflict detection

FieldSync scans assignments continuously. If an umpire is double-booked, a red warning and border appear directly on both affected game cards — no separate check is needed.

⚠ Jordan Smith is also assigned to AAA · MV Field 2 at 10:00 AM

To resolve: open the UMP dropdown on one of the conflicting games and choose a different umpire, or set to None and re-run Auto-Schedule.

Cancelled and postponed games are excluded from conflict checks.

Availability & time-off requests

Workers set their own availability in the app. Umpires only appear in the assignment dropdowns for dates they've marked available.

1
Open Requests in the nav — pending time-off appears here
2
Click Approve or Deny — the worker gets an in-app notification
3
Approved requests block Auto-Schedule on that date automatically
Always approve time-off requests before running Auto-Schedule so the engine works around them.
Section 4

Game Status & Rainouts

Game status

Change a game's status via the dropdown on each game card. Status affects pay and conflict detection.

StatusPay & Conflicts
ScheduledCounts toward pay; triggers conflict checks
CancelledExcluded from pay; no conflicts triggered
PostponedSame as cancelled until re-dated
RainoutClears all ump & crew assignments

⛈ Rainout tool

Cancels every game at one location on one date and clears all umpire and crew assignments in one action — faster than editing games individually.

1
Navigate to the affected date in Schedule view
2
Click the red ⛈ Rainout button next to that location
3
Confirm — all games marked Rainout and assignments cleared
Rainout cannot be undone. Umpire assignments are not recoverable. Use only when the full day is cancelled.

Section 5

Publishing & Reminders

Workers cannot see assignments until you publish. A green "Published" banner at the top of the week confirms it's live.

1
Fill all ump slots and resolve any red conflict warnings
2
Click ✓ Publish week
3
Umpires see their games in My Shifts immediately
4
ICS calendar subscriptions update within the hour
5
Click 🔔 Send reminders to push in-app notifications
Edits still save in real time after publishing. Use Send reminders to alert workers of any significant changes.
Section 6

Pay & Payroll Report

Go to Reports → Pay & Payroll. Umpire pay is calculated automatically: games assigned × rate per game. Cancelled and postponed games are excluded.

Setting pay rates

The Default pay rates panel sets the fallback for all umpires. For an individual override, type directly in the rate field next to that umpire's name in the pay table and tab away — saves instantly.

Exporting for accounting

1
Click ↗ Export for accountant
2
A formatted report opens in a new browser tab
3
Chrome: File → Print → Save as PDF

Quick Reference

✅ Weekly scheduling checklist
  • Import or add new games for the week
  • Approve or deny pending time-off requests
  • Navigate to the upcoming week in Schedule
  • Click ⚡ Auto-schedule — review assignments
  • Resolve any red conflict warnings
  • Click ✓ Publish week
  • Click 🔔 Send reminders
⚠ Common problems & fixes
  • UMP slot stays None after Auto-Schedule: no available umpire — check Staff → Workers availability
  • Umpire can't see their games: confirm the green "Published" banner is visible
  • Wrong pay total: verify game status is Scheduled and the correct rate is set
  • CSV rows skipped: dates must be YYYY-MM-DD, columns in required order
  • Dual-umpire game missing UMP 2: Today view flags it as "Missing ump"
